IMAGE CLASSIFIER V1.0 - Brandon Walsh

Python Dependencies
- Numpy
- Scipy
- Scikit-learn
- MatLibPlot

How to Run:

	cd project3
	python p3.py <filepath to test image>


The data file contains all images associated with this project. Inside the data file you will see the image classifications. Every classification folder must have the images classification name.

To test a file simply run p3.py with the images filepath as the first argument.
